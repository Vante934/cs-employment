计算机行业就业数据可视化
 项目介绍
这是一个基于Web的数据可视化项目，展示了计算机行业不同方向（前端、后端、人工智能、数据分析）的就业数据，包括城市分布、平均薪资、行业分布和热门岗位。
 功能特性
-  **数据可视化**：使用ECharts实现多种图表展示
-  **历年数据**：包含2020-2024年的就业趋势数据
-  **多维度分析**：城市、薪资、行业、岗位四个维度
-  **响应式设计**：适配不同屏幕尺寸
-  **数据爬虫**：自动从官方数据源和招聘平台爬取数据

 数据来源
- **官方数据源**：
  - 教育部大学生就业服务平台 (ncss.cn)
  - 麦可思研究院就业蓝皮书 (mycos.com.cn)
  - 工信部人才交流中心 (miitec.org.cn)
  - 清华大学、北京大学等重点高校就业质量报告
- **招聘平台**：
  - 拉勾网
  - 前程无忧
  - 智联招聘
  - BOSS直聘
  - 脉脉

## 技术栈
- **前端**：HTML5, CSS3, JavaScript
- **图表库**：ECharts 5
- **数据爬取**：Python 3, BeautifulSoup4, Requests
- **数据格式**：JSON

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/Vante934/cs-employment.git
cd cs-employment
```

### 2. 查看数据

直接打开 `index.html` 文件即可在浏览器中查看数据可视化效果。

### 3. 运行爬虫（可选）

如需更新数据，运行爬虫脚本：

```bash
python crawler.py
```

## 数据说明

- **城市分布**：基于各平台公开的招聘数据统计
- **薪资分布**：基于岗位薪资范围统计
- **行业分布**：基于企业行业分类统计
- **热门岗位**：基于招聘需求量排序

## 注意事项

- 数据仅供参考，实际就业情况可能因个人能力、地区差异等因素有所不同
- 爬虫脚本可能会因网站结构变化而需要更新
- 部分数据源可能需要科学上网才能访问
